How to make a stickman shape

0. Set up a grid (in Inkscape, File -> Document Properties...)
1. Draw lines of width 3, round join, round cap
2. Select all, convert strokes to path (in Inkscape, Path -> Stroke to Path)
3. Set stroke solid, width 1
4. Union
5. Paste head from another stickman icon
6. Turn off the grid
7. Adjust shoulders around head (manually delete/change Bezier curve handles)
8. Widen and drop points under armpits to thicken torso slightly
